# Unstable Studio Web Installer

The official web installer and landing page for **Unstable Studio**, a modern IDE built with Flutter for Android and Desktop.

## 🚀 About the Project

Unstable Studio is a high-performance, aesthetic, and functional IDE designed for professional coding on mobile and desktop devices. This web application serves as the primary distribution point and information hub for the project.

## 🛠 Features

- **Direct Download**: Easy access to the latest beta APK for Android.
- **Project Showcase**: Highlights key IDE features like the high-performance Code Editor, Integrated AI Agent (Gemini), and Terminal.
- **Modern UI**: A responsive, cinematic web experience built with React and Tailwind CSS.

## 💻 Tech Stack

- **Framework**: [React](https://reactjs.org/) (TypeScript)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Framer Motion](https://www.framer.com/motion/)

## 🛠 Development

### Setup
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
